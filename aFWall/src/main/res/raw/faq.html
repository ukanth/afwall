Q: Is there a Frequently Asked Questions (FAQ) page?
A: Yes, there is a FAQ page on GitHub.

Q: How can I send a bugreport to the developer?
A: On the application(s) list click on 'Firewall Rules' & 'Send error report' button.
This will send an Email to the developer (needs an active internet connection, make sure AFWall+ isn't blocked).

Q: The YouTube app is unable to connect?
A: Allow "Media Server" from the application(s) list.

Q: I see data traffic consumed by application xyz which is blocked by AFWall+, why?
A: It's possible false data traffic. AFWall+ uses IPtables which is at kernel level (low-level).
Usage data shown by Android is calculated before it gets dropped by firewall.

Q: Downloads are not working with stock Browsers/Google Play Store
A: Allow "Downloads" from the application(s) list.

Q: What's the difference between whitelist and blacklist?
A: Whitelist is for advance users where they know which app's can connect to internet including system application(s).
Blacklist is for beginners where they want to block only certain app's connecting to the internet.
It depends on person to person which mode fits best.

Q: What are all the icons on the top?
A: Local Area Network (LAN) - LAN data connections
Wifi - Wifi data connections
Mobile - 2G/3G/4G data connections
Roaming - data connections
Virtual Private Network (VPN) - data connections
Followed by 'revert' and 'clear' selection.
